Q. What are the primary differences between SQL and MySQL?

-> SQL
--SQL (Structured Query Language) is the standard programming language for managing and manipulating relational databases.
--The language provides a set of commands and syntax for creating, modifying, and querying databases.
--SQL is not tied to any specific database system and can be used with various database management systems (DBMS), including MySQL.

-> MySQL
--MySQL, on the other hand, is a popular open-source relational database management system (RDBMS) that uses SQL as its query language.
--It is one of the world's most widely used database systems and is known for its reliability, performance, and ease of use. MySQL provides a wide range of features for managing databases.
